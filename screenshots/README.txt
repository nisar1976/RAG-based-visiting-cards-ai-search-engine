Screenshots Directory
====================

Place your application screenshots here:

1. frontend-search.png    - Main search page with results
2. frontend-admin.png     - Admin panel showing all cards
3. backend-api-docs.png   - FastAPI Swagger UI at /docs
4. backend-health.png     - Health endpoint response

Capture Instructions:
- Start backend: cd backend && uvicorn main:app --port 8000
- Start frontend: cd frontend && npm run dev
- Open http://localhost:3000 for frontend screenshots
- Open http://localhost:8000/docs for API docs screenshot
- Open http://localhost:8000/health for health check screenshot

See SCREENSHOT_GUIDE.md in the project root for detailed instructions.
